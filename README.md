# Gender Predictor

A simple web application that predicts gender based on a given name using the Genderize API.

## About This Project

This project was developed as part of a community service workshop (IBM - Intermedia Bakti Masyarakat) to introduce high school students to basic web development concepts using HTML, CSS, JavaScript, and public APIs.

## Preview

Enter a name and click the **Predict** button to see the predicted gender and confidence level.

## Features

- Predict gender from a given name
- Display confidence percentage
- Modern and responsive UI
- Built with vanilla HTML, CSS, and JavaScript
- Uses the Genderize API

## Technologies

- HTML5
- CSS3
- JavaScript
- Genderize API

## How to Run

1. Clone this repository

```bash
git clone https://github.com/RizkyJulianto/gender-predictor.git
```

2. Open the project folder

3. Run using Live Server or open `index.html` in your browser

## API

This project uses the public API from:

https://genderize.io/

Example:

```text
https://api.genderize.io/?name=rizky&country_id=ID
```
